# Regras do Projeto

- Ao iniciar o projeto, os servidores devem ser iniciados manualmente pelo usuário usando o script `start_all.bat` ou conforme instruções abaixo.
- NÃO iniciar servidores automaticamente ao abrir uma nova conversa.
- SEMPRE que fizer alguma modificação ou atualização no código, faça o commit e o push automaticamente via git sem precisar pedir permissão prévia.
